## 使用说明

1. 打开 [Suno](https://suno.com) → Create → Custom Mode
2. 开启 **Instrumental**（纯音乐，无人声）
3. 粘贴对应的 **Style/Tags** 到 "Style of Music" 输入框
4. 填入 **Title**
5. 点击 Create

### 文件命名规范

生成后按以下格式命名，便于导入系统：
```
{序号}-{MBTI类型}-{能量等级}.mp3
例如：01-INTJ-low.mp3, 02-INTJ-mid.mp3, 03-INTJ-high.mp3
```

### 导入系统

生成完成后，将 48 个 mp3 文件放入 `server/data/audio-cache/fallback/` 目录，
然后更新 `server/data/fallback-manifest.json` 中的映射关系。
